# Lunar

Lunar is an enterprise-grade API management and AI control plane platform that provides a unified gateway for managing API traffic policies, rate limiting, quota enforcement, and API monetization across gateways. The platform combines an API consumption gateway with an MCP (Model Context Protocol) gateway to govern, secure, and observe interactions between AI agents, APIs, and data sources.

## Overview

Lunar delivers real-time visibility into model, API, and tool invocations alongside policy-based enforcement, cost controls, and advanced traffic shaping including retries, priority queues, and circuit breakers. It is recognized by Gartner as a Representative Vendor in both AI Gateways (2024, 2025) and MCP Gateways (2025).

## APIs

- **API Consumption Gateway** - Proxy-based gateway routing outbound HTTP/HTTPS traffic through YAML-configured flows for rate limiting, quota enforcement, and endpoint access control
- **MCPX Gateway** - Agent-native MCP gateway consolidating multiple MCP servers with authentication, scoped permissions, audit trails, and risk evaluation for AI workloads
